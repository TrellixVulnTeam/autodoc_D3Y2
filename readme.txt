Steps To Follow for Windows
1. Install PGMPy using following steps
	a. Git clone PGMPy using git clone https://github.com/pgmpy/pgmpy.git
	b. cd /path/to/pgmpy
	c. pip install -r requirements.txt
	d. sudo python setup.py install
2. Install Flask using
	pip install Flask
3. Install jsonpickle
	pip install -U jsonpickle
4. Execute the flask file using command
	python autodoc.py
5. To execute learning
	python datalearningtest.py
	
Steps to Follow for Linux/OSx

1. Initialize virtualenv
	source autoenv/bin/activate
2. Execute Flask
	sudo python autodoc.py
3. Execute learning
	sudo python datalearningtest.py
